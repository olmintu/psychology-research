# 🧠 Psychology Research Web App
**Cервис для проведения комплексных психологических исследований**

Этот проект представляет собой одностраничное веб-приложение (SPA), созданное для автоматизации сбора, подсчета и интерпретации данных в психологических исследованиях. Изначально разработанный для дипломной работы, проект вырос в полноценный модульный инструмент, который заменяет психологу ручную обработку бланков и построение профилей.

**[➡️ Попробовать вживую (Live Demo)](https://olmintu.github.io/psychology-research/)**

---

## 🎯 О проекте

Основная цель приложения — предоставить удобный, быстрый и автоматизированный способ проведения психологической диагностики. Система проводит респондента через все этапы: от заполнения анкеты до получения подробного отчета с интерпретациями, графиками и таблицами. 

Приложение работает полностью на стороне клиента (в браузере), что обеспечивает высокую скорость работы и конфиденциальность данных.

## ✨ Ключевые возможности

* 📊 **Комплексная диагностика:** Интегрированы три стандартизированные многофакторные методики.
* 🕵️ **Умная анонимность:** Возможность прохождения опроса без указания ФИО, но с "мягким запросом" (Soft Ask) социально-демографических данных (пол, возраст) для сохранения научной ценности.
* ⚡ **Моментальная обработка:** Сложные алгоритмы подсчета, перекрестные шкалы и выявление противоречий работают мгновенно по завершении теста.
* 📈 **Визуализация данных:** Наглядные графики мотивационного и эмоционального профилей, сравнение "Идеал / Реальность", диаграммы инновационного потенциала (на базе `Chart.js`).
* 💾 **Защита от потери данных:** Автоматическое сохранение прогресса в `localStorage` — если случайно закрыть вкладку, тест можно продолжить с того же места.
* 📥 **Экспорт и отчеты:**
  * 📄 **High-Res PDF:** Генерация красивого многостраничного отчета со всеми сырыми баллами, ключами, таблицами, графиками и подробными интерпретациями.
  * 📁 **JSON:** Скачивание сырых данных или загрузка ранее сохраненного файла для просмотра результатов.
* ☁️ **Облачная синхронизация:** Отправка результатов в Google Sheets (через Google Apps Script) для централизованного сбора базы данных исследования.
* 🌓 **UI/UX:** Адаптивный дизайн для ПК и смартфонов, поддержка Темной/Светлой темы, защита от "прокликивания" (Fast Clicker detection).
* 🛠️ **Режим разработчика:** Скрытая панель (активируется вводом `dev` в поле ФИО) для тестирования с автозаполнением всех 150+ вопросов случайными ответами.
* **Интерактивный ИИ-ассистент (Нейро-психолог):** Встроенный чат-бот на базе Google Gemini API. Ассистент получает структурированный JSON с посчитанными баллами и текстами интерпретаций, ведет осмысленный диалог с пользователем, помнит контекст беседы и строго ограничен рамками профессиональной этики (без постановки медицинских диагнозов).

## 📚 Интегрированные методики

Приложение включает в себя три методики для всестороннего анализа личности:

1. **«Методика исследования системы жизненных смыслов» (Б.С. Братусь)**
   * Оценивает иерархию и доминирование различных категорий жизненных смыслов (гедонистические, экзистенциальные, статусные и др.).
2. **«Диагностика мотивационной структуры личности» (В.Э. Мильман)**
   * Анализирует мотивационный профиль (поддержание жизнеобеспечения vs. развивающие мотивы).
   * Определяет эмоциональный профиль (стенические и астенические реакции в норме и в состоянии фрустрации).
3. **«Методика изучения особенностей проявления инновационного потенциала личности» (Ю.А. Власенко, В.К. Калина)**
   * Оценивает готовность человека к новому, стили реализации (поиск, оценка, действие) и аспекты взаимодействия с миром.

## 🏗️ Архитектура и технологии

Проект написан на **чистом (Vanilla) JavaScript**, HTML5 и CSS3 без использования тяжелых фреймворков. Код разделен на логические модули для удобства поддержки и масштабирования:

```text
📦 psychology-research
 ┣ 📂 css
 ┃ ┗ 📜 styles.css        # Стили, переменные тем, адаптив
 ┣ 📂 js
 ┃ ┣ 📜 app.js           # UI-логика, навигация, валидация, стейт-менеджмент
 ┃ ┣ 📜 data.js          # Тексты вопросов и ключи к методикам
 ┃ ┣ 📜 scoring.js       # Математика: подсчет баллов, определение типов, логика графиков
 ┃ ┣ 📜 pdf_export.js    # Верстка PDF-документа и рендер canvas-графиков
 ┃ ┗ 📜 sync.js          # Логика работы с JSON (экспорт/импорт/Drag-n-Drop)
 ┗ 📜 index.html         # Каркас приложения
```
## 🧠 Интеграция с нейросетью (Serverless Architecture)

Чтобы обеспечить безопасность API-ключей и конфиденциальность пользователей при хостинге на GitHub Pages, архитектура ИИ-чата реализована через паттерн **Proxy/Serverless**:

1. **Frontend (SPA):** Собирает объект `FULL_REPORT_CACHE` (баллы + текстовые выводы) и историю переписки, после чего отправляет POST-запрос на промежуточный скрипт. В интерфейсе реализован блок явного согласия пользователя (GDPR-friendly) на обработку обезличенных данных.
2. **Backend (Google Apps Script):** Выполняет роль безопасного прокси-сервера. Хранит  API-ключ, внедряет жесткий системный промпт для удержания LLM в роли психолога-консультанта и перенаправляет запрос к Google Gemini API.

**Сторонние библиотеки (подключаются через CDN):**
* [Chart.js](https://www.chartjs.org/) — отрисовка диаграмм и графиков.
* [pdfMake](https://pdfmake.github.io/docs/) — клиентская генерация PDF-отчетов.

## 🚀 Как запустить

### Вариант 1 — Онлайн (Рекомендуемый)
Просто перейдите по ссылке (сервер не требуется, всё работает в браузере): 
👉 **[olmintu.github.io/psychology-research](https://olmintu.github.io/psychology-research/)**

### Вариант 2 — Локально
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/olmintu/psychology-research.git
   ```
2. Откройте файл `index.html` в любом современном браузере.
   > **Примечание:** Для корректной работы экспорта графиков в PDF при локальном запуске рекомендуется использовать локальный сервер (например, расширение *Live Server* для VS Code), чтобы избежать ограничений CORS браузера на загрузку картинок.
